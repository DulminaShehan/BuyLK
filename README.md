# BuyLK

BuyLK is a production-oriented multi-tenant eCommerce platform for buyers, sellers, riders, and admins. The repository is structured to support a Flutter mobile client and a Node.js/Express backend with PostgreSQL and Prisma.

## Current Status
- Architecture and domain design documented.
- Flutter app skeleton created with Riverpod, GoRouter, and Material 3 theming.
- Backend skeleton created with Express and Prisma schema scaffolding.
- Authentication module scaffolded as the first production-ready feature slice.
